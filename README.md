# Tiddlywiki Plugin Files for Client-Server (Nodejs) Configuration

These are the [Tiddlywiki](https://tiddlywiki.com/) plugins installed on my
"Auxiliary Memory" tiddlywiki.

## Commander (plugins/kookma/commander)

### Summary

A tiddler toolkit for Tiddlywiki 5. Tiddler Commander can perform many kinds of
batch operation like create or delete tiddlers, rename tiddlers, add/remove
fields, tags, inspect tiddlers and do search and replace in all fields of
tiddler.

### Install

The plugin files were obtained from the
[kookma/TW-Commander](https://github.com/kookma/TW-Commander) github
repository. The plugin files were in the repository's `source/commander`
subdirectory.

Version: 2.1.14, Installed: 2026-03-03.

## ContextPlugin (plugins/danielo515/context-plugin)

### Summary

Add a new tab called 'Context search' to the 'Advanced search' tiddler,
displaying search matches with surrounding text

### Install

The plugin files were obtained and converted from the plugin creator's
[standalone tiddlywiki](http://contextplugin.tiddlyspot.com/).

The plugin is also available from the
[danielo515/TW5-TagSearch](https://github.com/danielo515/TW5-TagSearch) github
repository which contains the plugin files in the `plugins/tagSearch`
subdirectory. _The plugin files obtained from the github repository broke
Advanced Search completely,_ so the version from the standalone tiddlywiki was
preferred.

### Update

In accordance with these
[instructions](https://groups.google.com/d/msg/tiddlywiki/SA0Bm7SZG20/dBlE3CFvAAAJ),
replace occurrences of `tw-context` in plugin files with `tc-context`, e.g.,
the files `results` and `context.js`.

Version: 2.1.0, Installed: 2026-03-03.

## tagSearch (plugins/danielo515/tag-search)

### Summary

Adds a new tab called 'TagsSearch' to the 'Advanced Search' tiddler, enabling
search by tags alone, or by tags and tiddler content.

### Install

The plugin files were obtained and converted from the plugin creator's
[standalone tiddlywiki](http://braintest.tiddlyspot.com/).

Note that an preliminary step was required before following the 'obtain and
convert' process. Because the version of the standalone tiddlywiki was so old
(5.1.4) it was not possible to directly export the plugin as a json file. It
was necessary to create an empty tiddlywiki from the current release and import
the plugin into it by dragging and dropping. The 'obtain and convert' process
was used on the new tiddlywiki.

There is also a github repository for this plugin. The version from the
standalone tiddlywiki was used because it is newer than the repository,
although there is no guarantee the version of the plugin it contains is newer.

Version: 1.0.2, Installed: 2026-03-03.

## Utility Macros (plugins/dtn/utility-macros)

### Summary

This plugin provides the following macros:

- `check-var-or-macro` – check whether macro or variable exists
- `home-path-link` – display hyperlink to a path in user's home directory
- `list-item-space` – add vertical space after a list item
- `number-format` – format a number.

### Install

This repository is the primary source for 'dtn' plugins. The plugin files are
located in the repository's `plugins/dtn/utility-macros` directory.

Version: 0.0.2, Installed: 2020-09-22.
